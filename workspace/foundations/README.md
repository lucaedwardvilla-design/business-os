# Foundations

The core files that keep your day organized.

---

## Files to Create Here

### tasks.md
Your current task list:
- What you're working on this week
- Priorities and deadlines
- Backlog items

### gains.md
Your wins and progress:
- Completed tasks
- Achievements
- Progress notes

### daily-brief.md
Generated by `/morning`:
- Today's TOP 2 priorities
- Quick win for the day
- Context from recent work

---

## How It Works

1. Update `tasks.md` with what you need to do
2. Run `/morning` to get your daily brief
3. Work on your TOP 2
4. Log wins in `gains.md`
5. Repeat

---

## Example: tasks.md

```markdown
# Tasks

## This Week's Focus
Launch the new landing page

## Priority Tasks
- [ ] Write landing page copy
- [ ] Design hero section
- [ ] Set up email capture

## Backlog
- Update portfolio
- Create case study
- Record intro video
```

---

## Get Started

Create `tasks.md` with your current priorities. Then try `/morning`.
