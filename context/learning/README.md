# Learning & Research

This folder stores knowledge you want the AI to remember and reference.

---

## What to Add Here

### Organize by Topic

Create subfolders for different areas:

```
learning/
├── ai-tools/         # Claude, ChatGPT, automation
├── marketing/        # Content, growth, sales
├── business/         # Strategy, operations
├── productivity/     # Systems, workflows
└── {your-topic}/     # Whatever you're learning
```

### Types of Content

- **Video notes** - Insights from YouTube videos (use the YouTube skill)
- **Course notes** - Key takeaways from courses
- **Book notes** - Important concepts from books
- **Research** - Industry research, competitor analysis
- **Reference** - Materials you need to reference often

---

## How the YouTube Skill Uses This

When you process a YouTube video, the AI:
1. Extracts the transcript
2. Pulls key insights
3. Saves to the appropriate subfolder here

Example: A video about prompt engineering → `learning/ai-tools/prompt-engineering-tips.md`

---

## Why This Matters

This is your second brain for the AI.

When you ask about a topic, the AI can reference what you've already learned instead of giving generic answers.

---

## Example File

```markdown
# Advanced Prompt Engineering

**Source:** https://youtube.com/watch?v=...
**Type:** Tutorial
**Date:** 2024-01-15

## Key Takeaways
- Be specific about format and length
- Give examples of desired output
- Use system prompts for consistent behavior

## Actionable Items
- [ ] Create template library
- [ ] Test with different models
```

## Get Started

Don't worry about filling this out manually. Use the YouTube skill to process videos you watch, and this folder will grow naturally.
